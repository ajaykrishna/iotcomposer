Installation

In addition to the IoT composer distribution (see here) you need to install:

  • Java RE
    This is needed to transform IoT object models into the LNT formal specification.

    Download Java RE
    Note: IoT Composer requires JRE\JDK 1.8 and above.
  • Apache Tomcat Server 8.0
    This is needed for deploying and hosting the IoT Composer Web Archive (WAR)
     More information about tomcat deployment can be found here
    Download Tomcat 8.0

  • CADP toolbox
    This is needed to perform formal analyses on the formal description files.
    Download CADP

Browser Compatibility

The web app has been tested on the following browsers.

  • Mozilla Firefox Quantum 62.0
  • Microsoft Edge 42.17134.1.0
