Installation

IoT composer is distributed as a Web Archive (WAR). The project uses Apache Maven to generate WAR. The latest version of the WAR can be found here.

In addition to the IoT composer distribution (WAR), the following components are required for local installation:

  • CADP toolbox
    This is needed to perform formal analyses based on formal specification of the composition.
    Acquire CADP license

  • IoT Recommendation Service
    Used to get the initial device recommendation based on physical environment. It is a Bell Labs' solution. IoT composer connects to the service using the REST APIs provided by the service. More information about the platform can be found in the following paper
    IoT Recommendation Assistant

  • Majord'Home SDN-based Platform
    Bell Labs'platform for IoT deployments. IoT Composer connects to the platform using REST APIs. Composer uses APIs to create SD-LANs and enable communication interfaces. More information about the platform can be found in the following paper
    Majord'Home Platform

  • Once you have the appropriate license, please proceed to add the service URLs in the code. In the AppUtils file update the Recommendation service and Majord'Home URLs
